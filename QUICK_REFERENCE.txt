┌───────────────────────────────────────────────────────────────────────┐
│  📱 NEW PAGES                                                          │
└───────────────────────────────────────────────────────────────────────┘

  customer_order.php     → Customers order here (like kiosk)
  orders.php             → Cashiers process payments here
  barista.php            → Baristas update status here (unchanged)


┌───────────────────────────────────────────────────────────────────────┐
│  🔄 ORDER FLOW                                                         │
└───────────────────────────────────────────────────────────────────────┘

  Customer Orders        Barista Prepares       Cashier Gets Paid
  ───────────────        ────────────────       ─────────────────
  customer_order.php  →  barista.php         →  orders.php
  
  Status: pending     →  preparing/ready     →  completed
  Payment: unpaid     →  unpaid              →  paid ✓


┌───────────────────────────────────────────────────────────────────────┐
│  👥 WHO DOES WHAT                                                      │
└───────────────────────────────────────────────────────────────────────┘

  👤 CUSTOMER
     • Browse menu on customer_order.php
     • Add items to cart
     • Place order
     • Go to cashier when ready to pay

  💰 CASHIER
     • View orders on orders.php
     • Click "Process Payment" when customer pays
     • Take walk-in orders on neworder.php (still works!)

  ☕ BARISTA
     • View orders on barista.php
     • Update status: pending → preparing → ready

  👑 ADMIN
     • Can do everything


┌───────────────────────────────────────────────────────────────────────┐
│  💾 DATABASE CHANGES                                                   │
└───────────────────────────────────────────────────────────────────────┘

  orders table - NEW COLUMNS:
  
  customer_id      → Who ordered (NULL = walk-in)
  processed_by     → Which cashier processed payment
  payment_status   → unpaid / paid
  updated_at       → Last update time


┌───────────────────────────────────────────────────────────────────────┐
│  ✅ BENEFITS                                                           │
└───────────────────────────────────────────────────────────────────────┘

  ✓ Faster service (customers order themselves)
  ✓ Less cashier work (only handle payments)
  ✓ Better tracking (know who ordered & who processed)
  ✓ Modern experience (like Jollibee/McDonald's)
  ✓ Scalable (can add kiosks, mobile app later)


┌───────────────────────────────────────────────────────────────────────┐
│  📊 PAYMENT STATUS                                                     │
└───────────────────────────────────────────────────────────────────────┘

  🟠 UNPAID   → Customer hasn't paid yet
              → Shows "Process Payment" button
              → Cashier clicks when customer pays

  🟢 PAID     → Payment completed
              → No button shown
              → Order marked as completed
